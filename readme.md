# Project Selene

Project Selene is a Project Terra modloader that can be used in the browser or installed locally.

## Setup

To get started with Project Selene simply open [https://projectselene.org/](https://projectselene.org/) in Google Chrome or Microsoft Edge. If you do not use one of these browsers you can still visit the website to download and install it locally.

## Development setup

This project requires .NET 8 and NodeJS 20. Alternatively, one can use Docker using the Dockerfile in the root directory of the repository of this project to built it.

It is highly recommended to use a recent version of Visual Studio 2022 for C# development.
